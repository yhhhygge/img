# 向量筛选取子集

#### 1.建立一个向量

```R
x <- 8:12
```

####2.根据逻辑值取子集

```r
x[x==10]
x[x<12]
x[x %in% c(9,13)]
```

<img src="https://github.com/yhhhygge/img/img20200510173909.png" alt="逻辑值取子集" style="zoom:50%;" />

#### 3.根据位置取子集

```r
x[4]
x[2:4]
x[c(1,5)]#取第一个和第五个元素
x[-4]
x[-(2:4)]#反选取第一个和第五个元素
```

### 🌟修改向量中的某个/某些元素：取子集+赋值

x
